Xcode8注释失效

发表于 2016-10-29   |   分类于 工具   |     |  

每次升级Xcode对于插件来说都是一次灾难,总是会失效,这次Xcode8也不例外,不过类似VVDocument注释功能被苹果自己添加进来了,不用再用第三方的了

Xcode8自带的注释快捷键是

单行注释: command + /
多行注释: command + option + /

下图绿框中的为注释相关设置

如何修改快捷键

如果觉得默认注释快捷键不是自己喜欢的组合,则可以自己改,比如我把它修改成shift + / 其是就是 ?
